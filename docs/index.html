Features Checklist

  • Submit standard HTML <form> directly using POST.
  • Hono API Router parsing submissions 10x faster with RegExp Router.
  • Unlimited forms and submissions size under free Cloudflare tier.
  • Optional built-in Webhooks and email templates (Resend supported).
  • Active Honeypot fields filter automated bots.
  • Optional client Proof-of-Work (PoW) security challenge.
  • React 19 Hooks integration for instant dashboard responsiveness.
  • Self-Healing Database tables creation on first request.

Workflow Architecture

[ Pure React 19 Client SPA ] │ ▼ [ Hono Edge API Workers ] ──► (Spam HoneyPot / RegExp Routing) │ ▼ [ getDb() Call ] (Dynamic Connection Pool) │ ▼ [ Cloudflare D1 SQL Database ] (Store encrypted entries safely)

Important: FormForge Hono routes initialize database connections only during active handler execution using dynamic Drizzle pools. React static assets compilation never triggers D1 database access, ensuring zero database initialization crashes.

One-Click Deploy Flow

The **Deploy to Cloudflare** process is entirely decentralized. It clones the blueprint seed to your own GitHub profile, provisions your own Worker and D1 Database instances, and configures environment parameters directly under your ownership.

  1. Click the **Deploy to Cloudflare** button on the home page.
  2. Connect your GitHub profile to template clone the codebase.
  3. Define Worker target parameters (Worker name, e.g. formforge-backend).
  4. Secure your session keys by writing AUTH_SECRET (Generate keys from jwtsecrets.com).
  5. Click Deploy. The system configures variables, automatically builds server chunks, and deploys it live.

Variables Configuration

These environment keys can be modified from **Cloudflare Workers Settings -> Variables** panel anytime without redeploying the code:

Variable Name Required? Description
AUTH_SECRET Yes 32+ chars string to encrypt session cookies (dashboard security).
RESEND_API_KEY No Your Resend API token. If missing, email dispatchers remain inactive.
RESEND_FROM No Verified Resend sender domain. Example: FormForge <forms@domain.com>.
DB Yes Internal D1 database binder name. Auto-attached by Cloudflare.

FormForge gives you complete control over your user data, ensuring 100% data ownership.

Security Property FormForge (D1) Standard SaaS Providers

Pricing Limits Unlimited Free Tier Monthly submission limits (Paid upgrades)

Telemetry & Ads None Tracking pixels / Third-party cookies

Troubleshooting Errors

Error: Pbkdf2 failed: iteration counts above 100000...

Cloudflare Workers Web Crypto allows maximum 100,000 hashing iterations. In our updated lib files, this iteration limit has been set to 100,000. No further changes needed.

Error: Network error. Is the Worker deployed and D1 bound?

Verify that tables are created. Go to D1 Database -> Console tab on Cloudflare, and run D1 script to create the schema (Refer troubleshooting guides).
